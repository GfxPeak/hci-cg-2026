The Seven Stages of Action (Norman)

This model describes the psychological stages a user goes through to complete a task:

  1. Goal: Forming the intention.
  2. Plan: Identifying the action sequence.
  3. Specify: Mapping actions to physical movements.
  4. Execute: Performing the physical action.
  5. Perceive: Observing the state of the world.
  6. Interpret: Making sense of the perception.
  7. Evaluate: Comparing the outcome with the original goal.

GOMS Framework (Card, Moran, and Newell)

A theory of the cognitive steps required to perform a task. It stands for:

  • Goals: What the user wants to achieve.
  • Operators: The basic actions performed (e.g., press a key).
  • Methods: Sequences of operators to achieve a goal.
  • Selection Rules: Deciding which method to use when multiple exist.

KLM: Keystroke-Level Model

A low-level instance of GOMS used to predict the time it takes an expert user to perform a task without errors. It uses specific operators like K (Keystroke), P (Pointing), and M (Mental preparation).
